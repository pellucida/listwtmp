## listwtmp

List the logins recorded in a utmp(x) file.

```
listwtmp [-f wtmpfile] [-D since_days_ago ]
                       [-W since_weeks_ago ]
                       [-H since_hours_ago ]
                       [-M since_minutes_ago ]
                       [-S since_seconds_ago ]

```
